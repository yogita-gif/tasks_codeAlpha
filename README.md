# CodeAlpha Data Analytics Internship Tasks

## About
This repository contains the tasks completed during the **CodeAlpha Data Analytics Internship**. The projects focus on data analysis, data exploration, and data visualization using Python and Google Colab.

---

## Task 1: Exploratory Data Analysis (EDA)

### Objective
The objective of this task was to perform Exploratory Data Analysis (EDA) on a dataset to understand its structure, identify patterns, detect anomalies, and extract meaningful insights.

### Work Performed
- Imported and explored the dataset
- Checked data types and dataset structure
- Handled missing values
- Generated statistical summaries
- Analyzed relationships between variables
- Identified trends and patterns
- Created visualizations to support findings

### Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## Task 2: Data Visualization

### Objective
The objective of this task was to create effective visualizations that communicate insights and trends present in the dataset.

### Work Performed
- Created bar charts for category comparisons
- Generated line charts for trend analysis
- Built histograms to study data distributions
- Used scatter plots to analyze relationships
- Created heatmaps for correlation analysis
- Presented insights through visual storytelling

### Tools & Libraries Used
- Python
- Matplotlib
- Seaborn
- Google Colab
